# React

## Giới thiệu

```
ReactJS là một thư viện JavaScript nổi lên những năm gần đây với xu hướng Single Page Applicaion
Hướng đến mô hình MVC
```
## React là gì?

```
React là một thư viện UI phát triển bởi Facebook 
Xây dựng thành phần (component) UI có tính tương tác cao và có thể tái sử dụng lại được 
React được sử dụng là sản phẩm của Facebook

Không chỉ hoạt động client mà còn render trên server và có thể kết nối với nhau
React sử dụng khái niệm DOM ảo (Vitual DOM) để chọn lựa và render những phần tử của node dựa trên
sự thay đổi trạng thái khiến cho ta chỉ cần thay đổi ít thành phần nhất có thể giữ DOM update.
```

# SPA & MPA

## SPA - Single-Page Applicaion

```
- ReactJS là 1 trong những thư viện tạo ra SPA
- Các "ông lớn" sử dụng SPA: Google, Facebook, Twitter, ...
- Các SPA khác: F8, Shoppe, 30Shine, Chotot, Zing MP3, ...
```

## Cách triển khai 

```
- SPA - Single-Page Applicaion
- MPA - Multi-Page Applicaion
```

## Sự khác biệt 

```
SPA - Single-Page Applicaion | MPA - Multi-Page Applicaion
-|-|-
- Được cho là cách tiếp cận hiện tại hơn | - Là cách tiếp cận cổ điển hơn 
- Không yêu cầu tải lại trang trong quá trình sử dụng | - Tải lại trang trong quá trình sử dụng (Click đường link,...)
```

## So sánh 

```
SPA - Single-Page Applicaion | MPA - Multi-Page Applicaion
-|-|-
- Nhanh hơn khi sử dụng | - Chậm hơn khi sử dụng
- Phần lớn tài nguyên được tải trong lần đầu | Luôn tải lại toàn bộ trang khi truy cập chuyển hướng
- Trang chỉ tải thêm dữ liệu mới khi cần |
```

## Bóc tách 

```
SPA - Single-Page Applicaion | MPA - Multi-Page Applicaion
-|-|-
- Có phần Front-end riêng biệt | - Front-end và Back-end phụ thuộc nhau nhiều hơn, được đặt trong cùng 1 dự án
```

## SEO 

```
- SPA không thân thiện với SEO như MPA
- Trải nghiệm trên thiết bị di động tốt hơn
```

## Quá trình phát triển 

```
- Dễ dàng tái sử dụng code (component)
- SPA bóc tách FE & BE 
    + Chia team phát triển song song
    + Phát triển thêm mobie app dễ dàng
```

## Phụ thuộc JavaScript

```
SPA - Single-Page Applicaion | MPA - Multi-Page Applicaion
-|-|-
- Phụ thuộc hoàn toàn vào JavaScript | - Có thể không cần JavaScript
```
